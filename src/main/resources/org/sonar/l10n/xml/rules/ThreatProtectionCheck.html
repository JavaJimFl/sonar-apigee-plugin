A check for a body element should be performed before execution for policies JSONThreatProtection or XMLThreatProtection.

The condition check should be based on one of :

  • response.content
  • response.form
  • request.content
  • request.form
  • message.content
  • message.form
  • message.verb
  • request.verb
  • request.header.Content-Length
  • response.header.Content-Length
